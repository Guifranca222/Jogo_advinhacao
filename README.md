# Jogo do 0 a 100

Um jogo simples em Python que consiste em que o usuário adivinhe um número que o computador pensar de 0 a 100. 

---

##  Como Funciona
O programa utiliza a biblioteca "random" para gerar um número inteiro aleatório; 
O jogador deve inserir palpites até acertar, recebendo dicas do sistema:
* Se o palpite for maior que o número secreto, o jogo avisa:"Você errou! O número é menor.".
* Se o palpite for menor, o jogo avisa: "Errado! O número é maior.".
* Ao acertar, o jogo exibe o número de tentativas que você precisou.

---

##  Tecnologias Utilizadas
* Python 3.14
* Biblioteca "random" (nativa do Python)
